2.5 Executor

2.5.1 executor options and arguments

Usage

Usage: executor [options] <executor id>

options

-H < mbyte (k) >
Maximum heap size. The Unit of size is Mbyte. If "k" follows size, the unit is Kbyte(default 16).
-M <max>
Maximum number of threads(default 256).
-S <site id>
Site ID(default 1).
-T <ticks>
Number of ticks in one second.
-d
Running executor as a daemon.
-g
Debug mode(enable OzDebugf() output).
-i
Hide the global access monitor window.
-p
Enable the profile.
-s
Open executor shell window.
-v
Print the version of executor.
-a
Allow linking of all EXTERN symbols of executor.
-o
Open console window.

2.5.2 a note

SIGTERM
Termination of the executor
SIGINT
Create of the executor shell
SIGHUP
Forcible termination of the executor
