5.2 字句

OZ++ 言語の字句には以下のものがある。

  • コメント
  • 識別子
  • キーワード
  • 定数(整数、文字、浮動小数、文字列)
  • 演算子
  • 区切子

空白 (スペース、タブ、改行、改ページ) は字句の区切りとなる他は無視され る。

字句を読み込む場合には、構成可能な最長の文字列が字句として採用される。

5.2.1 コメント

OZ++ 言語では二種類のコメントがある。

  • /* で始まり */ で終るコメント

    この形式のコメントは入れ子にできない

  • // で始まり、改行で終るコメント

コメントは空白と同じ意味、すなわち字句の区切りとなる他は無視される。

5.2.2 識別子

英字から始まる英数字 (あるいはアンダースコア) の列であり、英字の大文字 と小文字は区別される。31 文字まで区別され、31 文字以上は長さは区別さ れるが、31 文字以上の内容が区別されることは保証されない。

また2つのアンダースコアにつづき[0-9a-fA-F]を16文字もつ識別子はクラス名 をクラス ID によって表現したものとして区別される. 

__0001000002000101      // 0001000002000101 というクラス ID のクラスを表す

5.2.4 定数

定数には以下のものがある。

  • 整数定数

    整数定数は数字の列である。

    • 0 から始まっている場合は 8 進数

      8 進数の場合は 8 と 9 を含まない

    • 0x あるいは 0X で始まる場合は 16進数

      16 進数の場合は a から f (大文字でも良い) を 10 から 15 として含む

    • それ以外の場合は 10 進数。

    u あるいは U がついた場合は unsigned を、 ll、 lL、 LL あるいは Ll がついた場合は long をあら わす. 

    123        // int型の10進数の123
0173       // int型の123(10進数)の8進数表記
0x76       // int型の123(10進数)の16進数表記
123U       // unsinged int型の123(10進数)

  • 文字定数

    文字定数はシングルクオートに囲まれた文字で、一文字を表す。一部の非 グラフィック文字、シングルクオート等は、表 5.2--2 に示すエスケー プシーケンスで表現可能である。

    　

    表 5.1--2: エスケープシーケンス
    newline NLL (LF)\n
    horizontal tab HT \t
    vertical tab VT \v
    backspace BS \b
    carriage returnCR \r
    formfeed FF \f
    audible alert BEL \a
    backslash \ \\
    question mark ? \?
    single quote ' \'
    double quote " \"
    octal number ooo \ooo
    hex number hh \xhh

    文字定数は char 型を持ち、その値はマシンの文字集合中 のその文字の数値である。 

    'a' '\n'        // 文字a、改行文字

  • 浮動小数定数

    浮動小数は整数部、小数点、小数部、 e あるいは E の後に指数 部が続き、それに続いてサフィックスが来る。サフィックスには

    • f あるいは F

    のがある。これは、

    • サフィックスがない場合は double

    • f あるいは F の場合は float
    を表す。 

    1.23 1.23E-1        // double型
1.23F 1.23E-1F      // float型

  • 文字列

    文字列は char 型の配列派生型である (5.7 節参照)。文字 列はダブルクォーテーションマークで囲まれた文字の列で、 非グラフィック 文字等は文字定数と同じエスケープシーケンスで表現される。ただし、文字列 の中に改行を含んではならない。

    となりあった文字列はコンカチーネートされ一つの文字列として扱われる。 

    "OZ++"
"OZ" "++"         // "OZ++" として

  • ユーザ定義の定数

    sharedの中で定数を宣言し、使用することが可能で ある (5.15.2 節参照)。

    ユーザ定義の定数は宣言された型を持つ。 

    float Pie = 3.14;        // float型の定数
int MaxSize = 100;       // int型の定数
char Name = "MyName";    // char の配列派生型の定数
