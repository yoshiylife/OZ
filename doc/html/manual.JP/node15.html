3.4 インスペクタの操作説明

名前

inspect - インスペクタ

形式

inspect [Local Executor ID]

機能

指定されたエグゼキュータ 上のグローバルオブジェクト あるいは、プロセス・スタックの内容を表示する。 同一ＷＳ上で実行されていないエグゼキュータ の場合、引数[Local Executor ID]で指定した エグゼキュータ を用いてアドレス解決を行うことにより、そのエグゼキュータ の グローバルオブジェクト あるいは、 プロセス・スタックのインスペクトが可能である。また、OZ++ におけるプロセスの意味に 従って、複数エグゼキュータ にまたがるプロセスのスタックの内容も表示することができる。 このコマンドにより表示されるウィンドウは、表示したい対象をダブルクリックすることにより より詳細な情報を表示するという方針に従って作成されている。（逆に、ダブルクリックしてもウィンドウが 現われない場合は、その機能が未サポートであることを意味する） 表示される主なウィンドウには次の種類がある。

  • Debugger
    このコマンドを実行すると最初に現われるウィンドウである。 インスペクトの対象とするエグゼキュータ のＩＤやアドレス解決に用いるエグゼキュータ のＩＤを 指定することができる。このウィンドウにある各ボタンの機能は次のようになっている。

    ボタン名 機能
    Object List Global Object List ウィンドウを表示する。
    Process List Process List ウィンドウを表示する。
    Quit 全てのウィンドウを閉じ、インスペクタを終了する。

  • Global Object List
    ウィンドウ（Debugger）の[Object List]ボタンを押すと現われるウィンドウである。 ウィンドウ（Debugger）で指定されたエグゼキュータ 上のグローバルオブジェクト の一覧を表示する。 このウィンドウにある各ボタンの機能は次のようになっている※。

    ボタン名 機能
    Refresh 表示内容を最新の状態にする。
    Inspect 選択されているグローバルオブジェクトの [Global Object Inspect]ウィンドウを表示する。 また、そのグローバルオブジェクトの状態を一時停止に する。
    Snap 選択されているグローバルオブジェクトのインスタンス変数 の内容を一端保存した上で[Global Object Inspect]ウィン ドウを表示する。この場合には、[Global Object Inspect] ウィンドウ表示中のグローバルオブジェクトの一時停止は 行われない。
    Status 選択されているグローバルオブジェクトのオブジェクトマネ ジャが保持している状態を表示する。
    Close このウィンドウ（Global Object List）を閉じる。

  • Process List
    ウィンドウ（Debugger）の[Process List]ボタンを押すと現われるウィンドウである。 ウィンドウ（Debugger）で指定されたエグゼキュータ を起点とするプロセスの一覧を表示する。 このウィンドウにある各ボタンの機能は次のようになっている※。

    ボタン名 機能
    Object List Global Object List ウィンドウを表示する。
    Refresh 表示内容を最新の状態にする。
    Inspect 選択されているプロセスの[Process Inspect]ウィンドウを 表示する。また、そのプロセスの実行状態を一時停止にする。
    Kill 選択されているプロセスを強制終了させる。
    Status 選択されているプロセスの実行状態を表示する。
    Close このウィンドウ（Process List）を閉じる。

  • Object Inspect
    オブジェクト の内容を表示する。 このウィンドウにある各ボタンの機能は次のようになっている。

    ボタン名 機能
    Refresh 表示内容を最新の状態にする。
    Inspect 選択されているオブジェクトの[Object Inspect]ウィンドウ を表示する。
    Process このグローバルオブジェクト（セル）を参照し、かつ、 プロセスの起点が当該エグゼキュータ上にあるプロセスの 一覧を表示する。
    Close このウィンドウ（Object Inspect）を閉じる。但し、 グローバルオブジェクトを表示しているウィンドウ （ウィンドウのタイトルがGlobal Object Inspectとなって いる）の場合は、このウィンドウから派生した全ての ウィンドウを閉じる。また、そのグローバルオブジェクトが 一時停止の状態である場合は、グローバルオブジェクトの状 態を一時停止の状態から開放する。

  • Process Inspect
    プロセス・スタックの内容を表示する。 このウィンドウにある各ボタンの機能は次のようになっている※。

    ボタン名 機能
    Inspect 選択されているオブジェクトの[Object Inspect]ウィンドウ を表示する。
    Kill 当該プロセスを強制終了させる。
    Close このウィンドウ（Object Inspect）を閉じる。当該プロセス の実行状態を一時停止の状態から開放する。

環境変数

OZHOME
OZ++ のホームディレクトリ

OZSITE_ID
サイトＩＤ

OZCLASSPATH
クラスのディレクトリ

注意

  • 引数[Local Executor ID]の指定では、１６進による指定の場合は先頭に 0x を付ける必要がある。
  • オブジェクト の内容を表示中は、そのオブジェクト が属するセル（グローバルオブジェクト ）は一時停止状態になる。但し、この一時停止の 状態は、オブジェクトマネージャ とは無関係に行われる。
  • プロセス・スタックを表示中は、そのプロセスは一時停止の状態になる。結果（排他制御等の影響）として、 他のプロセスに影響を与えることもある。
  • このインスペクタは、OZ++ システムが正常に動作をしていることを前提としている。従って、エグゼキュータ 自体が停止 ※ している場合、インスペクタはエグゼキュータ の再開を待つことになる。

例

inspect 0x101
#１６進で１０１のエグゼキュータ をアドレス解決のためのエグゼキュータ としてインスペクタを起動
